Anusaaraka

Anusaaraka derives its name from the Sanskrit word 'Anusaran' which means 'to follow'. It is so called, as the Anusaaraka output appears in layers i.e. a sequence of steps that follow each other till the final translation is displayed to the user. With the aim to reduce language barriers the Anusaaraka software allows the user to access text in a language that is not known to the user.

The layered output is unique to Anusaaraka. Thus, source language text information and how the Hindi translation is finally arrived at, can be accessed by the user. The important feature of the layered output is that the information transfer is done in a controlled manner at every step thus, making it possible to revert back without any loss of information. Also, any loss of information that cannot be avoided in a translation process is then done in a gradual way.

Tutorial

Here's how you can make the best use of the aids provided by the Language Accessor.


Search Lexical resources
1. Click+'d' on a word to get word meanings from shabdkosh.com.
2. Click+'g' on a word to get word meanings from glosbe.com
3. Click+'l' for local dictionary.

Search corpora for sentences
This utility helps in learning the usage of a word in different contexts.The keyboard shortcut is Click+'s'. Currently it displays sentences in both source and target language(the complete translated version).

Chat Utility
A user can connect to our IRC Channel Bhashaa_Setu to chat with other users.

Mail Utility
A user can mail an expert for more information and clarity. The keyboard shortcut is 'm' + Click. All the data related to a selected word like root, sentence etc, is automatically added to the mail.

Multi-Modal Utility
In this you can view the images and audio available for a selected word.The keyboard shortcut is Click+'v'.

Converter Utility
You can see a punjabi story in its transliterated version using this utility i.e. a story will be shown in target language script but not translated.
